渠道费用 -人力维护及折旧其中，营业额 = 使用频次×小时×单价×个数×出租天数用来电科技对外披露的数据来计算，每台充电宝每天被使用的频次在0.7~0.8次之间，每次租借时长约为三小时。如果以一个机柜内装40个充电宝来计算：每个机柜的月营业额=0.7~0.8 *3（小时）*40（台）*30（天） = 2520元~2880元一台机柜的实际利润=2520~2880 - 租金成本- 折旧费用其中，部分商场收取租金，部分不收租金，我们经过咨询，暂按照一线城市大型商超自动售货机（宽1.2米）场地租金价格来算，一个月一般为1000-2000元左右，但也有人说，在热门地段场地紧俏的时候，租金甚至高达5000、6000元。计算实际利润时，我们按照“租金+折旧”，每月共1500元来计算。*折旧（损毁）费计算方法：一台充电宝成本为100元/36个月的使用周期*40（台）=111元一台机柜的实际利润为=2520~2880 - 1500 = 1020~1380元/月而在硬件成本方面，充电宝+机柜成本在100元/个左右，每个机柜40个充电宝，成本为4000元。在人员维护成本上，据来电科技运营人员透露，一个维护人员可维护50个网点，200台设备，4000个充电宝的话，单个充电宝的维护成本在1.5元/月。回本周期=硬件成本+维护成本 / 利润 = 4000 + 1.5*40 / 1023~1380 = 约为3~4个月此外，我们将充电宝的损毁率折算到三年使用周期中，算出了年销售成本利润率：年销售成本利润率= 年利润 / 年成本 *100% = 月利润*12 / 年硬件成本 + 年维护成本 + 年租金= 63%~85%以上得出，仅通过租金收入，机柜式充电宝模式4个月便可回本，年成本利润率最高可达85%，同时，部分商家还可以通过出售数据线和机柜广告获得利润。腾讯创业在通过调查大型商超的广告机位租金发现，目前在一线城市的大型商超，单个广告机的广告费超过1000~2000元/月，同时部分商家还会出售数据线，盈利模式多样，且利润可观。据某投资机构的投资人透露，他们在投资前曾经做过测算，预计未来每个充电宝的回本周期在1~3个月。但是，能够得出这些利润的前提，主要是基于0.7~0.8的租赁频次，而这个数据均是通过共享充电的创业者们在公开场合所披露的数据。我们暂时无法验证其真实性。桌面充电的盈利模式桌面模式的盈利相对简单，主要以租金为主。据Hi电方面透露，Hi电设备的使用频次在3~5次/日，每次以1元/小时计算，一台设备一个月的营业额在90~150元之间。（桌面式充电）在租金方面，某桌面式充电模式的项目运营人员对腾讯创业说，他们目前合作的商家，都能提供免费的桌面租赁合作。鉴于市场上没有可参考的租金数据，我们暂时按照机柜式的单个充电宝月租赁成本，约30元/月进行计算。同时，我们把人力维护及折旧成本计算为10元/台。根据：利润 = 营业额 - 渠道费用 -人力维护及折旧利润=90~150-30-10=50~110（元）以每一台设备80~100元的成本计算（桌面式成本较低），除去人力成本与折旧，2个月便可回本。但从腾讯创业了解到的内部消息来看，不管是机柜式，还是桌面式，在进入商场时租金成本通常偏低，甚至许多商场和商家是免费进入，而实际的使用频率跟铺设的数量成比例关系。共享充电与共享单车三年盈利对比腾讯创业分别从：运营成本，使用年限，设备损毁率，回本周期，投与入产出，等五个方面与共享单车做了对比，从现金流的角度来说，共享充电无愧是一个“钱途光明”的生意。线下运营维护成本根据圈内人士提供数据显示，共享单车的维护目前是以区域划分，单车数量与维护人员的配比大概在1000：1，以一个维护人员6000元工资计算，一台单车的正常维护费用在6元/月。而共享充电的人员维护成本则相对较低，从上文中来电科技运营人员给到的数据来看，共享充电宝的维护人员配比大概在4000:1，单个充电宝的维护成本在1.5元/月。使用年限对比从共享单车披露的数据来看，理论上共享单车的使用寿命在3年左右。而共享充电宝的使用寿命，是以充电次数来计算，同时充电宝的循环使用次数在800次以上，以平均每天0.7~0.8次计算，寿命在三年左右。损毁率对比从腾讯科技曾经报道过的文章数据显示，摩拜单车的损毁率在10%，ofo的损毁率在20%-30% ，单车损毁后的调度费用每次便超过3元。而在共享充电领域，据来电科技相关人士透露，虽然其共享充电宝的使用寿命只有800次，但在中途损毁后，充电宝厂家会提供保修服务。对于共享充电的商家来说成本维修成本为0。回本周期对比据腾讯创业调查发现，共享单车一年约有270天有效骑行日；以每日2元收入，一辆共享单车一年可获得的总收入在500元上下，目前ofo的成本在400元，以ofo为例共享单车的回本周期在1年左右。而共享充电宝的回本周期，普遍在1个月~4个月。投入与产出对比以最便宜的共享单车500元为例，平均每月维修一次，其他时间正常维护，三年的投入成本为=400+12*3*（6+3）=718元，营业额约为1500元。而单个机柜式充电宝3年的投入成本为=100+36*1.5 =154元，3年租金约为1350元，合计1504元，在不计算广告的情况下，单个机柜式充电宝营业额约为2300~2600。根据腾讯创业的梳理后，你是否已经发现这是一个非常赚钱的行业呢？不管是人员维护费用，还是前期设备投入成本，都远远低于共享单车。那么，共享充电真的有这么赚钱吗？机柜式充电宝每天0.7~0.8的使用频率，桌面式 3~5次的使用频率，真的有如此高吗？真实情况到底如何，腾讯创业在上周走访了各大商圈的共享充电宝现场，如果想了解真实使用情况，请关注我们接下来的共享充电实地体验。同时，腾讯创业和来电科技CEO袁炳松聊了聊，袁炳松对腾讯创业表示，实际上，如果加上公司的运营成本，共享充电宝的年化收益也就在30%左右。此外， 随着战争的深入，免费充电未来也将会成为可能。那么，钱要从哪里赚？这是大家都在思考的问题。在系列报道的第四期，我们和几位创业者坐下来聊了聊关于共享充电融资背后的故事。推广：腾讯创业频道是腾讯旗下专注于创投领域的媒体平台。如果你想被获得千万网友和投资人的关注，即刻微信搜索qqchuangye关注我们，递交你的项目。
