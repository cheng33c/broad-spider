“Java之父”詹姆斯-高斯林腾讯科技讯 据外媒报道，亚马逊云计算部门（Amazon Web Services）的员工名册上增加了计算机科学领域内的一位重量级人物：业内公认的“Java之父”詹姆斯-高斯林（James Gosling）。高斯林周一在Facebook上宣布，他将作为杰出工程师加入亚马逊云计算部门。高斯林在Sun Microsystems工作期间提出了Java的最初设计，并且研制出第一台编译器和虚拟机。Sun在2010年被甲骨文收购，高斯林在公司被收购时离开，跳槽去了谷歌(微博)，但是只呆了很短的一段时间。在加盟亚马逊云计算部门之前，他在Liquid Robotics公司负责水下机器人的软件设计工作。亚马逊证实高斯林已经加入了公司，但没有对他的工作提供进一步的评论。与高斯林共事的还有很多技术大牛，他们包括：微服务架构专家阿德里安-考克罗夫（Adrian Cockroft）、人工智能副总裁斯瓦米-西瓦苏布拉马尼娜（Swami Sivasubramanian）和首席技术官沃纳-威格尔（Verner Vogels）。目前还不清楚高斯林将从事哪些具体的工作，但他似乎是AWS物联网工具团队的天然成员。他非常熟悉部署物联网系统的过程，以及使用公共云计算和物联网时面临的挑战。高斯林在2016年的知识产权博览会上说：“在Liquid Robotics工作时，没有云供应商可用，因此我们最终开发出自己的云计算基础设施，过程及其艰辛。我的意思是，很多云计算服务供应商会让我的生活变得更轻松，但是要想说服一名海岸警卫队员信任亚马逊却非常非常困难。”（编译/林靖东）
