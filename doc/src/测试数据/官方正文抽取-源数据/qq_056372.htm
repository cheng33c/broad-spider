BI中文站 5月15日报道现在看来，人们还没有用够Windows XP系统。近期攻击者利用Windows XP中的一个已知的重要漏洞发起了一次大规模网络攻击，引起了全世界的慌乱。虽然微软从2014年4月8日起就不再为Windows XP提供安全支持，但是现在不少PC仍在使用这款操作系统。据市场调查机构Net Applications公布的数据显示，其实Windows XP仍是全球第三流行的操作系统，它的市场份额为7.04%。这意味着，这款老旧的、微软不再支持的操作系统比任何版本的Windows 8、任何版本的Mac OS X和Linux更加流行。比Windows XP更流行的两款操作系统分别是Windows 7和Windows 10，它们的市场份额分别为49%和26%。据Net Applications称，目前最流行的5款操作系统分别为：Windows 7（48.5%）、Windows 10（26.28%）、Windows XP（7.04%）、Windows 8.1（6.96%）和Mac OS X 10.12（3.21%）。如果换算成具体数量，现在有多少台PC仍在运行Windows XP系统呢？市场研究公司Gartner之前曾估计，2014年时全球正在使用的PC总数约为20亿台，但是之后它就再也没有更新这个数据。按20亿台这个保守的数据来估算，这意味着目前仍有1.4亿台PC还在使用Windows XP系统。欧洲刑警组织Europol警告称，随着人们返回工作岗位并登录PC，被称为“WannaCry”的网络攻击本周将继续肆虐。WannaCry是一种勒索软件，这种恶意软件可以把人们的数据加密，然后要求对方付款以换取解密。据欧洲刑警组织称，到目前为止，全球150多个国家已有至少20万人受到这款勒索软件的攻击，并且导致英国国民医疗服务体系、西班牙电信和全球其他许多组织陷入了混乱。尽管微软已不再支持Windows XP，但它仍在周五晚间采取了不寻常的措施，针对Windows XP、Windows 8和Windows Server 2003发布了紧急补丁。（编译/林靖东）美国Business Insider作品的中文相关权益归腾讯公司独家所有。未经授权，不得转载、摘编等。微信公众号：BI中文站。
