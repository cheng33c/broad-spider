【TechWeb报道】6月2日消息，据国外媒体报道，三星预计将在韩国推出翻新的Galaxy Note 7作为Galaxy Note FE，韩国零售商现在列出该设备，并附有规格和定价细节。在韩国在线商店Danawa，Galaxy Note FE的价格为699,600韩元，约为625美元。这比原来的价格至少低200美元，但对比之前的Galaxy Note 7仍旧属于高价位。三星Galaxy Note 7由于电池不断爆炸，三星不得不将其从市面上撤下来，造成巨大的波动。此前，因为电池不安全问题，三星公司曾两次召回Galaxy Note 7，现在计划第三次推出。据悉，此次翻新的Galaxy Note 7改名为GalaxyNote FE，在韩国上市，“FE”的意思是“粉丝版”。在性能上，Galaxy Note FE预计与Galaxy Note 7几乎相同，除了一个主要的区别：电池。Galaxy Note 7具有3,500 mAh的电池，但Galaxy Note FE将会报出一个3,200 mAh的单元，应该能够安全避免过热和爆炸。从真机照来看，Note FE与Galaxy Note 7最大的区别在于官翻版背面底部多了一个大写的R符号，表示它是一台官翻机。据悉，手机应该在6月份打入商店。与此同时，三星已经推出了Galaxy S8和Galaxy S8+旗舰版，并且预计今年晚些时候推出性能更强大的Galaxy Note 8。推广：更多精彩内容，请微信搜索关注公众号「腾讯科技」（ID：qqtech）。
