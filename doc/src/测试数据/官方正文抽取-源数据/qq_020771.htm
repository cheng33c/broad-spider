腾讯科技讯 5月16日消息，联想集团董事长兼CEO杨元庆今日发表内部信宣布：联想中国区将重组为个人电脑及智能设备集团（PCSD）和数据中心业务集团（DCG）；刘军回归联想，将担任集团执行副总裁兼中国区总裁，领导中国平台及中国区PCSD业务；童夫尧 将担任集团高级副总裁兼中国区总裁，负责DCG中国的端到端业务，以及全球超大规模数据中心业务。以下是杨元庆内部信全文：各位同仁:中国区在公司整体业务中承担着举足轻重的角色，既是联想全球化的大本营，公司的现金牛，又是新业务的孵化器，新模式探索的急先锋，更是联想文化和价值观的源头，其发展对整个联想具有十分重大的意义。过去两年，在不断加剧的竞争下，中国区牢牢捍卫着PC业务的领导地位和强劲盈利能力，为全球PC业务在增长和利润间达到平衡做出了巨大贡献。在数据中心业务领域，中国区不断探索，逐步形成了清晰的业务模式，成为全球数据中心业务的主力军。同时，中国区还积极探索，孵化了不少新业务。对于公司全球的布局而言，中国区一直是坚强的中流砥柱。今天，正如我们都深切感受到的，PC不再只是传统的个人电脑，而是向新PC，即个人计算设备+个人云(Personal Computing + Personal Cloud)，乃至进一步向个性化计算设备+个性化云（Personalized Computing + Personalized Cloud）延伸。而中国是新一代智能设备发展变化最快的市场，并且全球创新中心也逐渐在向中国倾斜，在这里有着巨大的发展潜力和机会。因此，在充分发挥PC领域已经建立的核心竞争力的基础上，我们还需要明确更清晰的中国战略，建立更专注的管理和团队，打造更适应中国市场的创新产品和解决方案，进一步优化运营效率，以引领我们成功转型，迈向新的高度。在上个月的誓师大会上，我和大家分享了我们的三波战略，我们需要推动业务模式和组织模式的变革，来匹配战略的要求。针对PCSD业务和DCG业务在客户对象、销售能力和销售模式上的差异性，我们明确了各自面向客户的端到端的策略和组织。PCSD业务和DCG业务在中国都已经发展到了相当的规模，同时各自又都有巨大的潜力有待深耕挖掘和开拓创新。为此，从即日起，中国区也将重组为PCSD和DCG两个独立的组织，以便更专注地服务客户，更快速地响应市场。我非常高兴地宣布，刘军将回归联想，担任集团执行副总裁兼中国区总裁，领导中国平台及中国区PCSD业务，向兰奇汇报，并成为LEC成员。刘军是联想战功卓著的老将，他1993年加入联想，在产品开发、销售、企划等多种岗位上，屡创佳绩，是PC行业出类拔萃的领导者。联想著名的双模式业务的建立，就有他的直接贡献。依靠双模式业务，我们击败了强劲的竞争对手，建立起了独特的核心竞争力。刘军的战略视野、端到端丰富的经验，以及对PC行业的洞察，一定会带领中国区PCSD业务开拓新模式，实现新突破。同时，我宣布童夫尧将担任集团高级副总裁兼中国区总裁，负责DCG中国的端到端业务，以及全球超大规模数据中心业务，向Kirk汇报。童夫尧2005年加入联想，带领中国区大客户业务创造了辉煌的业绩。在过去2年半担任中国区总裁期间，童夫尧带领团队夯实了PC业务的领导地位，并大力投入数据中心业务，为企业级业务下一步在中国的全面发力奠定了坚实的基础。童夫尧对商用客户的深入洞察、对企业级市场的了解、卓越的精细化管理能力，以及超大规模数据中心业务在中国已经取得的重要市场地位，使得他成为这个新岗位的最佳人选。请大家和我一起，热烈欢迎刘军回归联想大家庭，衷心祝贺童夫尧履新，谱写新篇章！与此同时，怀着复杂的心情，我宣布现任联想集团高级副总裁、联想全球服务业务负责人陈旭东,出于个人和家庭的考虑，将于本月底离开联想。旭东是一名出色的领导者，在过去的24年中，他从一名应届毕业生一步一个脚印地成长起来，曾从事质量、服务、销售工作和负责手机业务。他早年在领导中国区服务业务时创造了用户满意度的最高记录，后来领导联想中国区实现了一轮又一轮的突破，他还曾成功启动了互联网创业公司神奇工场，最近又制定了全球服务的愿景和战略，为提高客户体验奠定了基础。我尊重旭东的个人选择，并祝愿他未来一帆风顺。从6月1日起，现任联想副总裁、PCSD全球服务业务负责人于洁，将领导联想全球服务业务，汇报给兰奇。于洁在服务领域的丰富经验和成功业绩将有助于全球服务业务再上新台阶。祝贺于洁！各位同事，未来三年是我们执行三波战略的关键时期，业务模式和组织架构的支撑和匹配对于我们坚决、全面地落实战略至关重要，相信通过这次调整，两支独立的业务团队能充分把握PCSD和DCG在中国的市场机遇，让中国区率先实现业务转型与变革，并成为新业务模式的先行军。
