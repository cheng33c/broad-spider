Surface Laptop 图自：staticworld微软在上个月连刷存在感，新产品线 Windows Laptop、新的 Surface Pro、Windows Creator 更新等软硬件悉数亮相，唯独绝口不提的是自家的 Windows Phone。不断下滑的市场占有率加上在任何场合几乎闭口不谈 Windows Phone 和 Windows 10 Mobile 的态度，让人觉得微软似乎要慢慢放弃手机业务。不过，事情似乎有了新的动向。据外媒 techradar 报道，微软似乎正在积极的开发新的 Windows 移动设备，这也意味着微软目前并没有放弃手机业务的打算。Lumia 950 XL 图自：alphr报道中还提到，微软不仅在测试新的硬件，还在开发和测试新的 Windows Mobile 操作系统。虽然没有关于这个新系统的具体细节，但据 techradar 表示，这将会是一个经过重新设计的系统。而另一家外媒 Thurrott 则表示，这个新的项目仍处于早期阶段，也就是说不排除未来会有较大的变数，但可以肯定的是，微软依然对移动设备保有很大的兴趣。传说中的 Surface Phone上个月，微软 CEO Satya Nadella（萨提亚·纳德拉）在一段采访中提到，微软并没有放弃手机业务的计划，他还表示微软接下来的移动设备将和“现在所有的手机完全不一样”。此外，纳德拉不断的在强调微软在 Surface 这种 2 in 1 设备上所获得的成功，或许微软也有将这种设计复制到未来手机上的可能，可传了很久的 Surface Phone 依然不见踪影。Lumia 950 和 Lumia 950 XL 图自：extremetech但 techradar 还表示，如果这项计划没有变化，新的硬件和新的系统最快可能也会在明年和大家见面，也就是说，今年依然不会有新的 Windows Phone 出现，而上一次微软发布新手机已经是 2015 年的事情了， 还是旗舰机型 Lumia 950 / 950 XL。根据长期跟踪微软动态的 Mary Jo Foley 表示，微软投资者关系总监 Chris Suh 告诉她，目前手机业务带来的收入基本为零。报告显示，截止到 2017 年 1 月份，美国地区 Windows Phone 手机占有份额是 1.6%，相比去年同期的 2.5% 再一次出现了下滑的状况。微软想在充满 iOS 和 Android 的世界里占有一席之地已几乎不可能，但将濒死的 Windows Phone 救活或许还有希望。推广：更多精彩内容，请微信搜索关注公众号「腾讯科技」（ID：qqtech）。
