（原标题：京城一理财公司疑“断链”丨“中财聚赢”发函称现金紧张无法兑付，已欠30余天房租） 5月23日，有投资者告诉重案组37号(ID:zhonganzu37)探员，称北京中财聚赢投资管理有限公司(简称“中财聚赢”)的投资项目到期无法兑换，涉事公司负责人也无法联系。探员当天探访发现，涉事公司办公场所已欠缴30余天房租，公司内只剩多名参与投资的员工在公司处理相关事宜。同日，重案组37号探员还多次尝试联系“中财聚赢”负责人的两个手机，一个提示停机，另一个提示关机。“中财聚赢”紧闭的大门。 新京报记者 曾金秋 摄街头频遇理财经理 利息最高16%!今年62岁的投资者李宁(化名)向重案组37号(微信ID:zhonganzu37)探员介绍，去年11月，他多次在双井地铁站附近遇到北京中财聚赢投资管理有限公司(简称“中财聚赢”)的客户经理，对方多次向他散发传单，并表示其公司有一项理财业务，半年利息13%。“他说让我帮帮忙，有多余的钱投一点，他们挣点业绩”。告诉他，投资期限可选择3个月、半年、一年，最高可获16%的利息。2015年11月，李宁与“中财聚赢”签订合同，投资15万。中财聚赢承诺，每月向他支付13%的利息，半年后返还本金加利息。对方还告诉他，到时间就准时支付利息及本金，“他们说，还不了就代赔偿。”“中财聚赢”每月向投资者李宁(化名)支付13%的高额利息。 新京报记者 曾金秋 摄到期无法兑现本金 副总突然失联?李宁表示，业务员向他承诺，今年5月到期归还本金及利息，但均未兑现。据“中财聚赢”向投资者出示《致投资人书》显示，“由于近期遭受持续性P2P行业存在高风险的舆论影响，以及公司业务副总的突然失联，导致众多客户欲提前兑付，且对公司业务产生一定的影响，致使公司近期现金紧张，无法正常兑付。”这份《致投资人书》还称，目前正核算全部资产及所有贷款产品的到期回款情况。待核算完成后，将在3个月内完成现有客户本金兑付。文件还称，“中财聚赢”已成立“兑付工作小组”，组长为一名姓闫的人士，重案组37号(微信ID:zhonganzu37)探员尝试拨打其电话，始终提示关机。业务员告诉李宁，共有100多名客户面临无法兑现的问题，“多的一两百万，少的几万、十几万。”“中财聚赢”向投资者出示《致投资人书》，称公司副总失联、现金紧张无法正常兑付。 新京报记者 曾金秋 摄出具担保函公司 企业系统查不到“中财聚赢”与李宁签订的《出借咨询与服务协议》中，双方商定的出借期限为半年。但在该公司出具的一份债权转让及受让协议中，“中财聚赢”将其债权转让给李宁，后者的钱最终被借给“河北东昌种业有限公司”。此外，在一份“聚赢集团国际发展有限公司”出具的一份担保函上，该公司称，“为保证申请人充分履行合同义务，应申请人的申请，本公司(聚赢集团国际发展有限公司)愿作为保证方出具此担保函。”探员在全国企业信用信息系统中查询发现，并没有名为“聚赢集团国际发展有限公司”的企业。“聚赢集团国际发展有限公司”出具的担保函。 新京报记者 曾金秋 摄涉事公司门紧闭 已欠房租30余天5月23日下午，重案组37号(ID:zhonganzu37)探员来到“中财聚赢”所在地，位于双井地铁站附近的佳龙大厦3层的一处办公地。该公司大门紧闭，屋内有工作人员商讨善后事宜。公司门口看到，有投资者手持报案材料等待消息，但不愿透露具体情况。该公司门口还张贴着北京金世纪大酒店有限公司5月20日贴出的一则通知，通知称，中财聚赢有限公司按规定应该于2016年4月6日前支付364087.5元租金，然而截至目前实际欠缴房租天数已达32天。通知称，根据我司5月5日催款通知，现我司认为贵司放弃租赁房屋内物品所有权，我司有权直接进行处置。“中财聚赢”门口张贴着租金欠缴和解除租赁合同的通知，显示该公司欠缴30余天房租。 新京报记者 曾金秋 摄公司多名员工参与投资 束手无策5月23日，原先负责联系李宁的客户经理告诉他，自己会帮助李宁追回钱款。该客户经理还称，自己的亲戚也投资了该项目，“我们也很着急。”据悉，该公司多名员工向亲朋好友推荐了该项目。李宁表示，去年签订合同时，一名业务员及其主管将自己的身份证复印件给他，让他放心。 “当时她告诉我，到时候要是钱没了，我赔你。”李宁称。一名员工告诉重案组37号(ID:zhonganzu37)探员，如今老板不在公司，他们也在焦急等待消息。 “中财聚赢”业务员和投资者商量追讨投资事宜。 新京报记者 曾金秋 摄涉事公司老板 两部手机均关机在“中财聚赢”出示的多份文件中，都提到一个叫黄海斌的人。业务员告诉李宁，这个人是公司的老板。重案组37号(ID:zhonganzu37)探员日多次尝试联系“中财聚赢”股东黄海斌的两个手机，一个提示停机，另一个提示关机。探员还曾尝试联系“河北东昌种业有限公司”办公电话，始终无人应答。 本文来源：新京报即时新闻 责任编辑：俞诗瑜_NN2555
