腾讯科技讯 据外媒报道，三星可能会将它召回翻新后的Galaxy Note 7改名为Galaxy Note FE，然后再重新出售。在2016年暑期，由于用户反映Galaxy Note 7爆炸着火，三星召回了这款手机，并停止了生产Galaxy Note 7。在今年3月，三星发表声明称，它将会“以对环境友好的方式”循环利用被召回的Galaxy Note 7。它还提到了将这款手机变成“翻新机或租赁机”。现在，据韩国新闻媒体报道称，这款手机的名称将会变更为Galaxy Note FE，其中FE是Fandom Edition的缩写，意思是“粉丝版”。尽管新的名称显得有点过于讨好它的粉丝，但是三星这样做可能并不是为了误导消费者。在今年1月，无线运营商Verizon称，现在有成千上万人仍在使用Galaxy Note 7。而三星报告称，93%的Galaxy Note 7已召回。韩国新闻媒体还指出，Galaxy Note FE将从6月底开始在韩国限量供应（估计只有30万部）。三星已证实，翻新的Galaxy Note 7不会销往美国。此外，还有报道称，翻新的Galaxy Note 7将会配置更小的电池，设定更低的价格。在召回Galaxy Note 7后，三星宣布执行八项电池安全检查措施，以确保其手机的安全。但是，我们并不清楚它的Galaxy Note 7翻新机是否也会经过这样的安全检查。（编译/乐学）
